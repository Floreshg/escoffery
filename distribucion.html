Agencias

La unidad de consumo comprende productos para uso masivo como: alimentos enlatados, productos de limpieza y hogar, bebidas, productos para mascotas, confitería.
Manejamos productos como atunes, sardinas, jamonillas, granos, agua, bebidas/jugos, fósforos, pastas, salsas, condimentos, gomas de mascar, pastillas, comida para perros/gatos, detergentes, desechables, entre otros.

La unidad de food service comprende productos para hoteles, restaurantes, cafeterías, servicios de catering, hospitales, fondas, entre otros.
Manejamos productos como pastas, salsas, encurtidos, aguas, bebidas especiales, galletas, quesos, jugos/néctares, vegetales, aceites, aceitunas, granos, vinos, cervezas, entre otros.

La unidad de veterinaria comprende productos para ganado vacuno, porcino y equino, para la avicultura; y para animales de compañía: perros, gatos, aves.
Manejamos productos como ungüentos, vitaminas, minerales, vacunas, medicinas, entre otros.

Los productos que manejamos son para ganado vacuno, porcino y equino, para la avicultura y mascotas. Dichos productos cuentan con vitaminas, minerales, vacunas, medicinas, ungüentos, entre otros.

La unidad de agencias comprende productos para la industria manufacturera de productos de acero, fábricas de colchones, plantas de irrigación.
Manejamos productos como bombas y arietes para agua, materia prima para velas, cables y productos de acero, cemento, alambres, entre otros.
